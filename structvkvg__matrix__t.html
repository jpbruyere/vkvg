Detailed Description

A vkvg_matrix_t holds an affine transformation, such as a scale, rotation, shear, or a combination of those. The transformation of a point (x, y) is given by:

x_new = xx * x + xy * y + x0;
y_new = yx * x + yy * y + y0;
